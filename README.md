## Installation

### 1. Setup Frontend

```
git clone https://github.com/khoa-anhtran/shopping-cart-app
pnpm install
```

```
// Add .env file

VITE_GOOGLE_CLIENT_ID=
VITE_API_BASE_URL=
```

```
pnpm dev
```

### 2. Setup Backend

```
git clone https://github.com/khoa-anhtran/shopping-cart-api
yarn install
```

```
// Add .env file

PORT=
MONGO_URI=
JWT_SECRET=
CORS_ORIGIN=
COOKIE_NAME=
NODE_ENV=
AZURE_AD_CLIENT_ID=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

```
yarn start:dev
```
